# 완벽한 세법 RAG를 위한 확인 질문 목록

> 출처: Codex(기술/인프라) + Gemini(도메인/UX) + Claude 코드 직접 검토  
> 중복 제거 후 카테고리별 정리. 답변 완료 시 `[x]` 체크.

## 1. 법령 커버리지

- [ ] 현재 Pinecone에 올라간 법령이 정확히 어떤 것들인가요?  
  _(현재 코드 기준: 소득세법 3종 + 조특법 3종만 수집됨)_
- [ ] 지방세법(지방양도세)은 판단 범위에 포함인가요, 제외인가요?
- [ ] 국세기본법(신고기한·가산세·경정청구)은 수집 범위인가요?
- [ ] 법령 이력은 몇 년도부터 수집됐나요? (최소 지원 과세연도)
- [ ] **별표(장기보유특별공제율 표1/표2)가 현재 Pinecone에 인덱싱돼 있나요?**  
  _(없으면 세액 계산 연동 시 답변 불가)_
- [ ] 부칙·경과규정이 본칙과 별도 청크로 분리돼 있나요, 아직 미수집인가요?
- [ ] 역사적으로 삭제·폐지된 조문이 해당 시기 케이스를 위해 보존돼 있나요?

## 2. FactInput 스키마 — 실제 누락 필드

### 날짜 관련
- [ ] **`acquisition_date`에 어떤 날짜를 입력하도록 안내하고 있나요?**  
  _(계약일 vs 잔금일 — 한국 세법 취득시기 = min(잔금지급일, 등기접수일). 사용자가 계약일을 입력하면 보유기간·조정지역 판단 오류)_
- [ ] `contract_date`(계약일)를 FactInput에 별도 필드로 받을 계획인가요?  
  _(DateBundle 내부에는 있지만 FactInput→변환 경로가 없음)_
- [ ] `balance_payment_date`(잔금지급일), `registration_date`(등기접수일)를 별도로 받을 건가요?

### 세액 계산 관련
- [ ] `necessary_expenses`(필요경비: 취득세·중개수수료·법무사비·자본적 지출 등)를 FactInput에 포함할 건가요?
- [ ] **동일연도 다른 자산 양도 여부**(`same_year_other_transfers`)를 받을 건가요?  
  _(같은 해 2건 이상 양도 시 세율 구간 누진 합산으로 실효세율이 달라짐)_
- [ ] `is_unregistered_transfer`(미등기양도 여부)를 받을 건가요?  
  _(해당 시 70% 단일세율 강제 적용)_

### 자산 분류 관련
- [ ] **오피스텔의 실제 사용 용도(주거/업무)를 어떻게 확인하나요?**  
  _(사업자등록 여부, 전입신고 여부 등 증빙이 필요한데 사용자가 직접 선택하는 구조인지)_
- [ ] 상가주택(겸용주택) 면적 비율 필드는 있는데, **구분 등기 여부**도 필요하지 않나요?
- [ ] `joint_ownership_ratio`(공동명의 지분율)를 받을 건가요?  
  _(boolean만 있고 비율 없음 — 고가주택 기준 적용 방식이 달라질 수 있음)_
- [ ] 무허가·미등기 건물(불법 증축 포함) 여부를 입력할 수 있나요?

### 세대 구성 관련
- [ ] 세대원 개별 주택 보유 내역이 필요한가요, 아니면 세대 합산 주택 수(`household_house_count`)로 충분한가요?
- [ ] **배우자·부모 등 세대원이 당해 연도에 별도 양도한 주택이 있는지** 확인하나요?  
  _(세대원 포함 합산으로 중과 판단이 바뀔 수 있음)_
- [ ] 사실혼 관계(주민등록 미등재 배우자)를 처리하나요?
- [ ] 주민등록상 세대와 실제 생계 세대가 다른 경우를 처리하나요?  
  _(예: 자녀가 부모 집에 등재돼 있지만 타 도시 거주)_

### 농어촌주택 특례
- [ ] 농어촌주택의 읍·면 소재 여부(`rural_house_location`)를 입력할 수 있나요?  
  _(조특법 §99의4 특례 — 소재지 확인 필수)_

## 3. 상수·버전 관리 — 코드에서 발견된 버그성 이슈

- [ ] **`HIGH_VALUE_THRESHOLD`(고가주택 기준)가 12억으로 하드코딩돼 있는데, 2020년 이전(9억 기준) 케이스를 지원할 계획인가요?**  
  _(연도별 버전 관리 없으면 과거 케이스 오답)_

  _(2025년 이후 계약이 오류 처리될 수 있음)_
- [ ] 다주택 중과배제 유예 종료일(`HEAVY_TAX_SUSPENSION_END = 2026-05-09`)이 현재 법령 기준으로 맞나요?
- [ ] 장기보유특별공제율 표1/표2가 개정되면 어떻게 반영할 계획인가요?

## 4. 조정대상지역 자동 조회

- [ ] **`admin_notices.py`에 `MOEF_API_KEY`가 미발급 상태인데, 조정대상지역 이력 API 발급 계획이 있나요?**

## 6. 도메인 경계 — 무엇을 "판단한다"고 정의하나요?

- [ ] **부담부증여**(양도세 + 증여세 동시 발생)는 처리 범위인가요?
